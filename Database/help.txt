
Projectary database tester suite.

Usage: npm run bd-test [options] [script] [volume]

Options:

-h				Outputs this page, also if no options selected.
-lt				Lists available DB tables tests
-lp				Lists available DV procedures tests
-T 				Tests all tables
-P				Tests all procedures
-A				Tests everything
-ts [script]	Tests specific tables script
-ps [script]	Tests specific tables script

[volume]		Batch size/Number of iterations. Default is 2.
